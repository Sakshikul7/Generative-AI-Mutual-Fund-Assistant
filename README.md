# Generative AI Mutual Fund Assistant

## Project Overview

This project is an AI-powered Mutual Fund Assistant built using Generative AI concepts.

The assistant helps users with:
- Mutual Fund Queries
- SIP Information
- Investment Suggestions
- Portfolio Guidance
- Market Insights

## Features

- Prompt Engineering
- Retrieval-Augmented Generation (RAG)
- Conversational AI
- Financial Query Handling
- Market Overview System

## Tech Stack

- Python
- Flask
- Hugging Face Transformers
- GPT-2
- VS Code

app.py              # Main API file
